# AWS EC2 Web Server Deployment

## Overview
This project demonstrates hands-on experience in deploying and managing a web server on AWS EC2 using Linux and Nginx. The focus is on applying cloud fundamentals with real-world infrastructure practices.

## Key Highlights
- Launched and configured an EC2 instance on AWS
- Managed security groups and SSH access
- Installed and configured Nginx on Linux
- Deployed a static website
- Documented the complete setup clearly for repeatability

## Tools & Technologies
- AWS EC2
- Linux (Ubuntu/Amazon Linux)
- Nginx
- Git & GitHub

## Deployment Steps
1. Create and configure an EC2 instance
2. Set up security groups for HTTP and SSH access
3. Install and start Nginx web server
4. Deploy website content
5. Verify access via public IP

## Outcome
Successfully hosted a web application on AWS EC2, gaining practical exposure to cloud infrastructure setup, Linux server management, and basic DevOps workflows.
